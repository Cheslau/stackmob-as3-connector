del()method
public function del(domain:String, params:Object = null):void

Deletes objects.

Makes a DELETE request for deleting objects in the specified domain as a parameter (i.e. "users").

It will fire a flash.events.Event.COMPLETE when there's a result available. The result, in json format, will be stored in the data attribute.

In other case, it will fire a flash.events.IOErrorEvent.IO_ERROR event. The exact error will be stored in the data

login()method 
public function login(username:String, password:String):void

Authenticates the user against StackMob REST API.

It'll check if the credentials are valid looking for provided username and password exist inside users schema.

If a succesful login occurs, it will fire a com.cycleit.stackmobconn.event.LoginEvent.LOGIN event.

In other case, it will fire a com.cycleit.stackmobconn.event.LoginEvent.LOGIN_ERROR event.

Parameters

username:String — Username
 
password:String — Password
